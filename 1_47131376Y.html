Definir un array per desar objectes JSON. En concret els objectes han de tenir l'estructura: 

                obj = {
                    [param1: value,]
                    [param2: value,]
                    [param3: value,]
                    sortida: value
                        }
Assignar valors als objectes de manera manual, però només als camps param1, param2 i param3 que representaran als possibles paràmetres d'entrada dels jocs de prova indicats en la taula anterior.

Tot seguit fer un recorregut per l'array dels objectes, invocar el nou mètode i desar el què retorni dins del camp sortida.

Finalment, mostrar dins l'etiqueta HTML amb id test el valors de cadascun dels objectes, però es vol mostrar un únic objecte cada 2 segons. Quan ja no quedin més objectes a mostrar s'atura el procés.
Els objectes de tipus Date s'han de mostrar amb el format dd/mm/yyyy.
